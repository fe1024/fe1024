lipper examples

lipper 轻量的涟漪效果库

用法：

指定选择器，支持querySelector 选择器详情见文档

lipper.init({
  duration: 1.5, // 涟漪持续时间，单位秒
  selector: ['button', '.container'], // 选择器，字符串或字符串数组
  radius: 50, // 扩散半径
  zindex: 1000,
  center: true, // 是否在容器中央涟漪
  overflow: false, // 是否在容器外继续涟漪
  color: 'rgba(255, 0, 0, .5)' // 涟漪的背景色
})
